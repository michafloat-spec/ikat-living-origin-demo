# IKAT Living Origin Demo

Separate mobile-first customer demo for IKAT Living Origin.

This project is intentionally independent from the IKAT Monitoring System. It uses local mock data and local assets only. No Google Sheets, Apps Script, upload services, or live database are required.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output in `dist/` is GitHub Pages friendly because Vite is configured with a relative base path.
